Tweakpane provides a text field for a number property by default.

var params = {
  speed: 0.5
};
var pane = new Tweakpane();
pane.add(params, 'speed');

Range

Use min or max option to specify minimum/maximum value. 
var pane = new Tweakpane();
pane.add(target, 'speed', {min: 0});

If you specify both minimum and maximum value, slider control will be appear.

var pane = new Tweakpane();
pane.add(target, 'speed', {min: 0, max: 100});

Stepping

You can also specify a step of changing value by step option.

var pane = new Tweakpane();
pane.add(target, 'particleCount', {
  min: 0, max: 100, step: 10
});

List

If you want to choose a value from named values, pass an object that has number values as list option.

var pane = new Tweakpane();
pane.add(target, 'direction', {
  list: {
    none: 0,
    horizontal: 1,
    vertical: 2,
    both: 3
  }
});
