Range

Use min() or max() to specify minimum/maximum value. 
var ss = new Seasoner();
ss.add(target, 'someProperty').min(0);

If you specify both minimum and maximum value, slider control will be appear.

var ss = new Seasoner();
ss.add(target, 'someProperty').min(0).max(100);

Stepping

You can also specify a step of changing value by using step().

var ss = new Seasoner();
ss.add(target, 'someProperty').min(0).max(100).step(10);
