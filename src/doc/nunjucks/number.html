addControl() provides a text field for a number property.

var params = {
  speed: 0.5
};
var pane = new Tweakpane();
pane.addControl(params, 'speed');

Range

Use min or max option to specify minimum/maximum value. 
var pane = new Tweakpane();
pane.addControl(target, 'speed', {min: 0});

Slider

If you want to use a slider control, use addSlider() instead and specify both minimum and maximum value.

var pane = new Tweakpane();
pane.addSlider(target, 'speed', {min: 0, max: 100});

Stepping

You can specify a step of changing value by step option.

var pane = new Tweakpane();
pane.addSlider(target, 'particleCount', {
  min: 0, max: 100, step: 10
});

Selector

If you want to choose a value from named values, use addSelector() and pass an object that has number values as values option.

var pane = new Tweakpane();
pane.addSelector(target, 'direction', {
  values: {
    none: 0,
    horizontal: 1,
    vertical: 2,
    both: 3
  }
});
