1. Import Tweakpane

First, download the latest Tweakpane and import it to your web page.

<script src="tweakpane-{{ config.version }}.min.js"></script>

2. Create a pane

Create an instance of Tweakpane by Tweakpane().

<script>
var pane = Tweakpane();
</script>

3. Ready to go!

Add components to fine-tune a property, monitoring a property, etc. See the reference on the left side for more detailed information.

// ...

var pane = Tweakpane();
pane.add(target, 'color');
pane.monitor(target, 'speed');

// ...
