Adding a folder

Use addFolder() to add a collapsible folder. It returns a instance of added folder, so you can add all types of property by using its add() just like other properties.

var ss = new Seasoner();

var folder1 = ss.addFolder('Folder1');
folder1.add(target, 'foo');
folder1.add(target, 'bar');

var folder2 = ss.addFolder('Folder2');
folder2.add(target, 'baz');
folder2.add(target, 'qux');
To close a folder, use close(). 
var ss = new Seasoner();

var folder1 = ss.addFolder('Folder1');
folder1.add(target, 'foo');
folder1.add(target, 'bar');
folder1.close();

var folder2 = ss.addFolder('Folder2');
folder2.add(target, 'baz');
folder2.add(target, 'qux');
