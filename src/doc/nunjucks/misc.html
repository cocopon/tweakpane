Changing a label of a property

Property name will be used as a label by default. To change it, use label option.

var pane = Tweakpane();
pane.add(target, 'initSpeed', {
  label: 'Initial speed',
  min: 0.0,
  max: 1.0
});

Placing a pane in a specified element

You can specify a container element by container option.

var pane = Tweakpane({
  container: document.getElementById('someElementId')
});
